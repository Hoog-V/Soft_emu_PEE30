Installeer een UART terminal (putty, terraterm, e.g.)

Als je al een UART terminal app hebt geïnstalleerd op je computer, kun je deze stap overslaan. Het gebruik van putty wordt aangeraden. Terraterm werkt ook maar kan vrij onstabiel zijn soms.

Download putty hier

Kies voor de optie: 64-bit x86 MSI installer en installeer het op je pc.

 

Hier een video als je er niet uitkomt...

Voor een grafische uart terminal:

Open de terminal en tik in: sudo apt-get install putty

Voor een uart sessie in de terminal:

Open de terminal en tik in: sudo apt-get install screen

Voor een grafische uart terminal:

Open de terminal en tik in: sudo pacman -S putty

Voor een uart sessie in de terminal:

Open de terminal en tik in: sudo pacman -S screen

Open de applicatie terminal In de terminal kunnen we straks een UART sessie openen met het screen commando.

Let op!

Hiervoor moet je homebrew op je mac hebben geïnstalleerd

Open de terminal en tik in: brew install putty

Als die vraagt om een wachtwoord moet je het inlogwachtwoord invoeren.
