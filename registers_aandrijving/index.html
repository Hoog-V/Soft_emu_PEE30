Start [1:1] Start aandrijving

Sinit [2:1] Start initialisatie

Mogelijke waarden zijn 1 of 0.

Een waarde van 1 geeft aan dat de aandrijving moet starten met de test of initialisatie.

Een waarde van 0 geeft aan dat de test of initialisatie nog niet gestart is.

RO= Read only

RW= Read/write
